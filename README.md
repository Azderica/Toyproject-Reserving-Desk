### 패키지 구조

- components : 컴포넌트
- router : 라우터
- store : 상태 관리
- api : api 함수
- utils : 필터 등의 유틸리티 함수
- mixins : 믹스인
- plugins : 플러그인
- translactions : 다국어
- assets : css 등의 웹 자원
